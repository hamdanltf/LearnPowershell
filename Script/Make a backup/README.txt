##To use the script, right-click on backup.ps1 select run or you can run directly from power shell

##if you want to excluding specific file or directory, you can add the list on exclude-folder.ps1, to add the list right click on exclude-floder.ps1 open with notepad or your favorite text editor (you can follow the example provided on the file)

##If you don't want certain file's extension(s) to be copied, you can add it on exclude-files.ps1, to add the list right click on exclude-floder.ps1 open with notepad or your favorite text editor (you can follow the example provided on the file) WARNING! this feature will delete all file(s) with the certain extensions you input! do it carefully!
